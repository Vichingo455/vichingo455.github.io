# Come installare macOS Big Sur
## Prima di iniziare
* [Scarica VMware Workstation](https://vichingo455.github.io/VMware/)
* [Scarica VMware macOS Unlocker](https://vichingo455.github.io/VMware%20macOS%20Unlocker/)
* Installa VMware Workstation e macOS Unlocker (istruzioni nella pagina di download)
* Scarica macOS Big Sur ISO (16 GB) in RAR e estrai i file (pasw se richiesta: *vichingo455.github.io*)
